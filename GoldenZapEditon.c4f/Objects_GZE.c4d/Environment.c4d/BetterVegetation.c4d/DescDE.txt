Liest die "Vegetation Anteil"-Liste in den Umwelt Eigenschaften des Szenarios aus um damit wesentlich mehr und wesentlich besser Vegetation zu platzieren. 

Erlaubt auch wesentlich mehr Einstellmöglichkeiten, wie die Vegetation plaziert wird. Folgende Optionen können im Script.c des Vegetationsobjekt stehen:


// Ist einge hängende Pflanze (steht nicht auf dem Material sondern hängt wie z.b. Lianen)
public func IsHangingVegetation() { return(0); }

// Diese Pflanze wächst nur im Untergerund (1) oder nur auf der Oberfläche (0)
public func IsUndergroundVegetation() { return(0); }

// Diese Pflanze wächst in einer Flüssigkeit z.B.: Material("Water") or Material("Lava"). Ansonsten 0 für Platzierung ausserhalb von Flüssigkeiten.
public func IsSubmergedVegetation() { return(0); }

// Pflanze wächst nur auf diesem Material. Bei 0 wächst sie auf allen Materialien mit Einstellung Soil=1 (z.b. Erde) in der .c4m
public func GetVegetationSoil() { return (Material("Grass")); }

// Platzierungs-Mulitplikator. Grass zum Beispiel wird doppelt so oft platziert. Auch sehr nützlich um an Szenarienspezifisch an der Platzierung zu schrauben (mittels #appendto)
public func GetVegetationPlacementBalance() { return(200); }

// Wie tief sind die Wurzeln dieser Pflanzen? (Wird x Pixel ins Material versetzt)
public func GetVegetationRootDepth() { return(5); }

// Hier kann ein Winkelbereich in Grad angegeben werden, wenn sich die Pflanze abhängig von der Landschaft drehen soll, ansonsten 0. Beispiel Bäume: Dürfen sich von -30 bis +30 Grad drehen.
public func GetVegetationRotates() { return [-30, 30]; }

// Hier kann zufällige Wachstumsfortschritte in % angegeben werden. Der Wert wird intern auf 1 - 100 begrenzt, somit kann mit einer Angabe von z.b. [50, 200] erreicht werden, dass die Platzierung voll ausgewachsener Bäume höher ist. Bei 0 werden Pflanzen immer vollausgewachsen platziert.
public func GetVegetationCons() { return [5, 200]; }

Vorsicht: Entfernt zu Begin den Engine-Platzierte Vegetation. Scriptplatzierte Vegetation mit den gleichen IDs daher erst ein 1-2 Frames später erzeugen!
